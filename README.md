## ✨ Features

- 📂 **Upload any tabular file**  
  Supports **Excel (.xlsx, .xls)**, **CSV**, and **TXT** files.

- 🧠 **Smart header detection**  
  Automatically detects the correct header row even if the file has titles or extra rows.

- 🗂️ **Auto column mapping (Sales mode)**  
  Tries to detect:
  - `Product / Item / Name`
  - `Category / Group`
  - `Quantity`
  - `Unit Price`
  - `Total Sale / Revenue / Amount`

- 🧮 **Two intelligent modes**
  - **Sales Mode:** When product + sales columns exist  
    → Shows *highest-selling* & *lowest-selling* products, revenue KPIs, category-wise charts.  
  - **Generic Mode:** For any random table  
    → Automatically picks a **label column** and a **numeric column**, and still shows ranking, charts, and stats.

- 📈 **Interactive visualizations**
  - Pie chart (category-wise or label-wise share)
  - Bar chart for **top N items**
  - Clean KPI summary (total revenue / rows / unique products, etc.)

- 💡 **Smart suggestions**
  - Highlights revenue concentration
  - Low-performing products/categories
  - Basic guidance for next business/data steps

- 🌐 **Runs anywhere**
  - Perfect for **Google Colab**
  - Also works locally with plain Python
